# TP3 AEDS3 - Casamento de Padrões

## Compilação do programa

```cmd
make
```

##  Execução do Programa

```cmd
./tp3 input.txt <estratégia> 

Estratégia 1 = Shift And
Estratégia 2 = Força Bruta
Estratégia 3 = KMP

```

### Compilação e Execução Manual

```cmd
gcc -o tp3 ./main.c
./tp3 input.txt <estratégia> 

Estratégia 1 = Shift And
Estratégia 2 = Força Bruta
Estratégia 3 = KMP
```
